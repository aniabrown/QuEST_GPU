1 # QUEST
2 
3 A library for simulating operations on sets of qubits.
4 
5 Operations implemented:
6 * Rotations of a single qubit
7 * TODO: Measure if a single qubit is in the zero state
8 * TODO: Two qubit phase gate

10 ## Getting started
11 
12 With git: clone the root directory to your device
13 
14 git clone https://github.com/aniabrown/QUEST.git [QUESTHOME]
15 
16 Without git: download the directory manually
17 
18 'Clone or download' > 'Download zip'
19 
20 In the root directory, run a simple 8 qubit example on one node:
21 
22 The default compiler is the gnu compiler. To change to the intell compiler, edit the COMMON CONFIG section at the beginning of makefile in the root directory. Change COMPILER=GNU to COMPILER=INTEL. Then run:
23 
24 ```
25 make clean
26 make
27 ./demo 8
28 ```
29 
30 This will report some information about the size of the system, perform rotations and verify that
31 the state vector of probability amplitudes is still normalized.
32 
33 OpenMP is enabled by default. To change the number of threads used by the program, set the value of OMP_NUM_THREADS before running the program, eg export OMP_NUM_THREADS=16. See examples/ompJob.sh for an example of how to do this in a job submission script.

35 ## Building other examples
36 
37 There are other examples of codes using the QUEST library in the examples folder. To use one of these,
38 copy from the examples folder into the root folder, eg:
39 cp examples/timingDemo.c timingDemo.c
40 
41 Edit the COMMON CONFIG section at the beginning of makefile. Change MY_FILE_NAME to the name of the file without
42 any extension, eg MY_FILE_NAME=timingDemo
43 
44 Change the name of the executable as desired eg EXE=myProg.
45 
46 Run with:
47 ```
48 make clean
49 make
50 ./myProg [NUMBER OF QUBITS]
51 ```
52 To run on arcus-b on one node, use the job submission script examples/ompJob.sh
53 
54 ## Creating a new file from the template
55 
56 The initial example file available in the root folder is the template file basicTemplate.c. If you have removed
57 this file, it is also available in examples/basicTemplate.c. The structure of this file is:
58 
59 ```
60 Initialisation
61 Rotations
62 Measurement
63 2 Qubit phase gate
64 Cleanup
65 ```
66 
67 In general, leave the initialization and cleanup sections and edit the rotations, measurement and phase gate
68 sections. Further explanations are in the template file. The full list of QUEST library functions that can be used here is listed in qubits.h.
69 
70 ## Multi node code
71 
72 To run over several nodes with MPI, edit the COMMON CONFIG section at the beginning of makefile.
73 
74 Change USE_MPI=0 to USE_MPI=1.
75 
76 Run with:
77 ```
78 make clean
79 make
80 mpirun -np [NUMBER OF PROCESSES] ./demo [NUMBER OF QUBITS]
81 ```
82 To run on arcus-b, use the job submission script examples/mpiJob.sh
83 
84 Note that the API to the QUEST library is unchanged when running on multiple nodes. The template file basicTemplate.c
85 is valid for both single and multi node environments.
